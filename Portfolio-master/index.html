HTML

My first skill set which I love so much. HTML is the standard markup language for creating Web pages. I use HTML5 markup for my projects. It is the latest evolution of the standard that defines HTML. It reduces development time. I feel like this helps the focus on the functionality of my projects and websites turn out looking really clean, polished, and fresh.

CSS

I have style for days. I really enjoy working with style. CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once. I have learned to make my layouts responsive building for mobile first.

Javascript

This is where the magic happens. You can really get things elevated and dynamic with this programming laungage. JavaScript is an object-oriented computer programming language commonly used to create interactive effects within web browsers. This is a must know to get a strong foundation set for React.

React

I have recently began working on web applications using React. In the coding world, React (also known as React.js or ReactJS) is a JavaScript library for building user interfaces. It is maintained by Facebook and a community of individual developers and companies. React can be used as a base in the development of single-page or mobile applications.This is pretty amazing and has helped improve the way I code in JavaScript so it is a winner for me.
